Galley is for React apps that need Markdown editing without a separate Markdown-to-HTML preview pane. It uses CodeMirror 6 and Lezer's Markdown parser to hide syntax marks when the cursor leaves a node, then reveal raw Markdown when the user edits that node.

<CardGrid>
  <Card title="Live preview, not HTML conversion">
    Markdown remains the document format. Galley decorates the editor view instead of maintaining a second rendered document.
  </Card>
  <Card title="Designed for product surfaces">
    Bring your own toolbar, upload pipeline, renderer hooks, and CSS tokens while keeping the core editor controlled.
  </Card>
  <Card title="Extensible at CodeMirror depth">
    Use built-in renderers, register custom plugins, or add CodeMirror extensions through the public API.
  </Card>
</CardGrid>

## Current Scope

Galley is currently `v0.10.0` and still pre-1.0. The documented API is the intended public surface for the current package, but minor releases may still refine behavior before the first stable release.
