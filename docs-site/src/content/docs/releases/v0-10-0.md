v0.10.0 release notes.

v0.10.0 is the public GitHub and npmjs transition release for Galley Editor.

## Highlights

- Renamed the package to `@inkyquill/galley-editor`.
- Moved the canonical public repository to GitHub.
- Switched release automation from GitLab to GitHub Actions and npmjs.
- Enabled GitHub Pages for Astro docs and Storybook under one public site.
- Kept the publishable library output unchanged: `dist/index.js`, `dist/index.d.ts`, and `dist/style.css`.

## Verification

The transition passed:

- `npm run lint`
- `npx tsc -b`
- `npm run test:coverage`
- `npm run build:lib`
- `./scripts/verify-publish.sh`
- `npm run build-storybook`
- `GALLEY_DOCS_SITE=https://inkyquill.github.io GALLEY_DOCS_BASE=/galley-editor npm run docs:build`
