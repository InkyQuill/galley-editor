v0.8.0 is the asset-workflows milestone for Galley Editor.

## Highlights

- Added consumer-owned file paste and drop handling through `onFiles`.
- Added `onFileStatus` and `input.report()` so apps can observe upload progress while Galley's editor-resident placeholders and optional overlays stay responsive.
- Added hybrid upload interaction modes: inline placeholders, drop indicators, overlay mode, and locked mode.
- Added `onFileError` for upload handler failures.
- Expanded `imageRenderer` metadata with `url`, `title`, `width`, `height`, raw source, preserved attrs, and source positions.
- Added Markdown image metadata syntax: `![Alt](image.png "Title"){width=640 height=360}`.
- Added `updateImageMetadata` and `clearImageDimensions` commands for asset inspectors, alongside built-in visual image resize handles.
- Added missing-image placeholders with a `missingImageRenderer` override.
- Added Storybook integration examples for paste upload, drop upload, upload placeholders and overlays, missing-image placeholders, image metadata editing, and image resize/source reveal behavior.
