v0.8.0 is the asset-workflows milestone for Galley Editor.

## Highlights

- Added consumer-owned file paste and drop handling through `onFiles`.
- Added `onFileStatus` and `input.report()` so apps can show upload progress while Galley keeps the editor responsive.
- Added `onFileError` for upload handler failures.
- Expanded `imageRenderer` metadata with `url`, `title`, `width`, `height`, raw source, preserved attrs, and source positions.
- Added Markdown image metadata syntax: `![Alt](image.png "Title"){width=640 height=360}`.
- Added `updateImageMetadata` and `clearImageDimensions` commands for asset inspectors and resize controls.
- Added Storybook integration examples for paste upload, drop upload, image metadata editing, and a command-backed resizable image renderer.
