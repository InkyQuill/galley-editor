v0.9.1 is a consumer integration polish release based on the first production-style Galley integration.

## Highlights

- Added `ariaLabel` for naming the underlying CodeMirror editing surface without a consumer extension.
- Documented public LLM-readable Pages URLs for `llms.txt` and the complete Markdown guide.
- Expanded customization docs for Remix/React icon replacement, dense workspace surfaces, `theme="auto"` in host apps, stable styling hooks, read-only surfaces, and app-owned save/dirty workflows.
- Kept v0.9 table editing behavior unchanged.
