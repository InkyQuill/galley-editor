v0.9.0 is the table UX milestone for Galley Editor.

## Highlights

- Visual editing for simple GFM pipe tables.
- Excel-lite cell navigation with `Tab`, `Shift+Tab`, `Enter`, arrow keys, `Esc`, and single-cell paste.
- Row, column, alignment, delete, and Source controls on rendered tables.
- Markdown source remains the source of truth.

KaTeX and Mermaid renderer examples are deferred to v0.10.
