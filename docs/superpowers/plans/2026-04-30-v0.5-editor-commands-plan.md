# v0.5 Editor Commands Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the v0.5.0 command milestone: line editing commands, navigation/search helpers, exported default keymap metadata, and `docs/commands.md`.

**Architecture:** Commands live in focused modules under `src/commands/editing/` and `src/commands/navigation/`, with `src/commands/index.ts` remaining the public registry. `EditorController` consumes exported `DEFAULT_KEYMAP` and preserves the existing compartment-based keymap reconfiguration.

**Tech Stack:** TypeScript, CodeMirror 6 `EditorView`/`EditorState`, Vitest, React wrapper tests, Markdown docs.

---

## Resolved Spec Questions

- Slugs use a GitHub-like subset: lowercase, trim, remove punctuation except spaces/hyphens, collapse whitespace to `-`, and append `-1`, `-2`, etc. for duplicate headings.
- `swapLineUp` / `swapLineDown` collision rule is lower-numbered cursor wins; conflicting later ranges no-op.
- `Mod-D` defaults to `duplicateLine`.

## File Map

- Create `src/commands/editing/duplicateLine.ts`: duplicate touched lines and move selections to duplicated text.
- Create `src/commands/editing/sortSelectedLines.ts`: sort selected line blocks ascending/descending, optional whole-document mode.
- Create `src/commands/editing/swapLine.ts`: swap touched lines up/down with collision protection.
- Create `src/commands/editing/insertLine.ts`: insert blank lines before/after touched lines.
- Create `src/commands/navigation/jumpToHash.ts`: heading slug scan and cursor/scroll jump.
- Create `src/commands/navigation/findInDocument.ts`: plain/regex/whole-word search helper.
- Modify `src/commands/index.ts`: export new commands, names, navigation helpers, and `DEFAULT_KEYMAP`.
- Modify `src/controller.ts`: use `DEFAULT_KEYMAP` for controller keymap assembly.
- Modify `src/types.ts`: extend `BuiltinCommand`, add find types.
- Modify `src/components/index.ts`: export new command helpers and types.
- Create `docs/commands.md`: command reference.
